Build a backend for listing items that have not yet expired.

Write this schema to `convex/schema.ts`:
```ts
import { defineSchema, defineTable } from "convex/server";
import { v } from "convex/values";

export default defineSchema({
  items: defineTable({
    name: v.string(),
    expiresAt: v.number(),
  }).index("by_expiresAt", ["expiresAt"]),
});
```

Create a query `listActive` in `convex/index.ts` that:
- Takes arguments `{ now: number }`, where `now` is a Unix timestamp in milliseconds supplied by the caller
- Returns an array of at most 100 item documents whose `expiresAt` is strictly greater than `now`, ordered by ascending expiration time (soonest-expiring first)
